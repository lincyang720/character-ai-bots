# YouTube 视频制作指南

## 📸 第二步：获取素材

### 1. 网站截图（最重要）

**工具选择：**

**方法 A：浏览器截图（推荐）**
- **Chrome/Edge：**
  - 按 F12 打开开发者工具
  - 按 Ctrl+Shift+P（Mac: Cmd+Shift+P）
  - 输入 "screenshot"
  - 选择 "Capture full size screenshot"
  - 自动保存完整页面截图

**方法 B：截图工具**
- **Windows：** Snipping Tool（Win+Shift+S）
- **Mac：** Cmd+Shift+4
- **第三方：** Lightshot, ShareX（免费）

**需要截图的页面：**
```
1. 首页：https://www.characteraibots.com/
2. 搜索页：https://www.characteraibots.com/search
3. 分类页：https://www.characteraibots.com/category/yandere
4. 角色详情页（10个）：
   - https://www.characteraibots.com/characters/yandere-classmate
   - https://www.characteraibots.com/characters/vampire-teacher
   - https://www.characteraibots.com/characters/tsundere-nurse
   - ... 等等（选择评分最高的 10 个）
```

**截图技巧：**
- 使用 1920x1080 分辨率
- 确保页面完整显示
- 避免个人信息
- 截图前清理浏览器标签

### 2. 背景音乐

**免费音乐网站：**

**YouTube Audio Library**（推荐）
- 网址：https://studio.youtube.com → Audio Library
- 完全免费，无版权问题
- 分类：Tech, Upbeat, Ambient
- 推荐曲目：搜索 "Tech" 或 "Electronic"

**其他免费音乐网站：**
- **Pixabay Music：** https://pixabay.com/music/
- **Free Music Archive：** https://freemusicarchive.org/
- **Incompetech：** https://incompetech.com/
- **Bensound：** https://www.bensound.com/

**推荐风格：**
- 轻快的电子音乐
- 科技感背景音乐
- 音量：低（不要盖过旁白）

### 3. 音效

**免费音效网站：**
- **Freesound：** https://freesound.org/
- **Zapsplat：** https://www.zapsplat.com/
- **Mixkit：** https://mixkit.co/free-sound-effects/

**需要的音效：**
- 过渡音效（whoosh）
- 点击音效
- 开场音效
- 结尾音效

### 4. 字体和图形

**免费字体：**
- **Google Fonts：** https://fonts.google.com/
- 推荐：Roboto, Open Sans, Montserrat

**图标和图形：**
- **Flaticon：** https://www.flaticon.com/（免费图标）
- **Canva：** https://www.canva.com/（免费设计工具）

### 5. Logo 和品牌元素

**创建简单 Logo：**
- **Canva：** 免费 Logo 制作
- **或使用：** 🤖 emoji 作为临时 Logo

## 🎥 第三步：录制旁白

### 方法 A：自己录音（推荐）

**工具：**
- **Windows：** Audacity（免费）
- **Mac：** GarageBand（免费）
- **在线：** Online Voice Recorder

**录音技巧：**
1. 使用耳机麦克风或手机
2. 在安静环境录音
3. 说话清晰，语速适中
4. 录制多次，选择最好的

**录音步骤：**
1. 下载 Audacity：https://www.audacityteam.org/
2. 打开脚本
3. 按录音键
4. 读脚本
5. 导出为 MP3

### 方法 B：AI 语音（快速方案）

**免费 AI 语音工具：**
- **ElevenLabs：** https://elevenlabs.io/（免费额度）
- **Play.ht：** https://play.ht/（免费试用）
- **Natural Reader：** https://www.naturalreaders.com/

**使用步骤：**
1. 注册账号
2. 复制脚本文字
3. 选择英文语音（推荐：美式英语）
4. 生成音频
5. 下载 MP3

## 🎬 第四步：视频制作

### 方法 A：简单方案（推荐新手）

**工具：CapCut（完全免费）**
- **下载：** https://www.capcut.com/
- **平台：** Windows, Mac, 手机
- **优点：** 简单易用，免费，无水印

**制作步骤：**

1. **导入素材**
   - 打开 CapCut
   - 点击 "New Project"
   - 导入所有截图
   - 导入旁白音频
   - 导入背景音乐

2. **创建时间轴**
   - 将截图按顺序拖到时间轴
   - 每张截图显示 45 秒
   - 添加旁白音频
   - 添加背景音乐（音量调低至 20%）

3. **添加效果**
   - 截图之间添加过渡效果（Fade, Slide）
   - 添加文字标题（角色名称、评分）
   - 添加缩放效果（Ken Burns effect）

4. **添加文字**
   - 开场：视频标题
   - 每个角色：名称 + 评分
   - 结尾：网站链接 + 订阅提示

5. **导出视频**
   - 分辨率：1920x1080（1080p）
   - 帧率：30fps
   - 格式：MP4
   - 质量：高

### 方法 B：专业方案

**工具：DaVinci Resolve（免费专业软件）**
- **下载：** https://www.blackmagicdesign.com/products/davinciresolve
- **优点：** 专业功能，完全免费
- **缺点：** 学习曲线较陡

### 方法 C：在线工具

**Canva Video Editor**
- **网址：** https://www.canva.com/video-editor/
- **优点：** 在线使用，简单
- **缺点：** 免费版有限制

**Clipchamp**
- **网址：** https://clipchamp.com/
- **优点：** Windows 内置
- **缺点：** 功能较少

## 📤 第五步：上传到 YouTube

### 1. 创建 YouTube 频道

1. 登录 YouTube
2. 点击右上角头像
3. 选择 "Create a channel"
4. 设置频道名称：例如 "AI Character Guide"
5. 添加频道描述和头像

### 2. 上传视频

1. 点击 "Create" → "Upload video"
2. 选择视频文件
3. 填写信息：
   - **标题：** Top 10 Character AI Bots for Roleplay in 2026
   - **描述：** 使用准备好的描述
   - **缩略图：** 使用 Canva 制作（1280x720px）
   - **标签：** 添加所有标签

4. 设置：
   - **观众：** 不是儿童内容
   - **年龄限制：** 无
   - **类别：** 教育或科技

5. 发布：
   - 选择 "Public"
   - 点击 "Publish"

## 🎨 缩略图制作

### 使用 Canva（免费）

1. 访问 https://www.canva.com/
2. 搜索 "YouTube Thumbnail"
3. 选择模板
4. 修改文字：
   - 大标题："TOP 10"
   - 副标题："Character AI Bots"
   - 添加："2026"
5. 添加网站截图
6. 下载（1280x720px）

**缩略图技巧：**
- 使用大字体
- 高对比度颜色
- 包含人脸或角色图片
- 添加数字（TOP 10）
- 保持简洁
